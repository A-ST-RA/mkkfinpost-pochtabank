# Vanila Web Starter

Стартер приложения на ванильном стеке `html5` + `css3`

## Для чего

Нужен для создания лендингов, маленьких веб страниц

## Как работать

1) Установить пакеты `yarn`

2) Запустить приложение `yarn start:dev`
